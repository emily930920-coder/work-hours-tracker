# 工时记录系统

一个简单易用的工时记录系统，支持打卡、统计和数据导出。

## 功能特点

- ⏰ 上下班打卡
- 📊 本月统计（工作天数、总工时、平均工时）
- 📖 历史记录查看
- ✏️ 手动添加/编辑记录
- 📥 数据导出/导入（JSON格式）
- 💾 本地存储（无需服务器）

## 使用方法

直接在浏览器中打开 `index.html` 即可使用。

数据保存在浏览器本地存储中，不会丢失。

## 部署到 GitHub Pages

1. 在 GitHub 上创建一个新仓库
2. 上传 `index.html` 文件
3. 在仓库设置中启用 GitHub Pages
4. 选择 `main` 分支作为源
5. 访问 `https://你的用户名.github.io/仓库名/index.html`

## 数据管理

- 点击"导出数据"可以下载 JSON 文件作为备份
- 点击"导入数据"可以从备份文件恢复数据
- 数据格式为 JSON，可以手动编辑

## 技术栈

- 纯 HTML + CSS + JavaScript
- 无需后端服务器
- 使用 localStorage 本地存储
